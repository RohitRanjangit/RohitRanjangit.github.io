P2P video Conferencing App

The project comprises of developing a simple P2P video-chat web application using python and WebRTC. This project introduced some topics like Python, Git, Flask (a micro web framework written in Python) and WebRTC.

  • Rating:
  • Members: 8
  • Website: yet to host
  • Completed:1 Apr 2019

This application help users to communicate with other registered users in this app via video conferencing as well as personal video chat.In first week of this project we read documentation of flask micro-framework and got little familiarity of git, github as well.we were working into five sub-groups where each group had to build some specific part of application. My group was working on User registration,login and logout part.Our team included me, Mudit Agarwal and Sandeep Sharma. we built our part in 1 and half month using flask micro framework and a little use of css, html and javascript.Later we worked on communication part.In second week we made git repository and commited initial commits of user registration,login and logout.

In next week we did some work on communication part of application. we implemented backend system of communication using WebRTC (a javascript library) where servers of two users communicate through sending of data file in JSON format.In next week we implemented conferencing system of multiple users, where more than two users can share their ideas collectively.Later, we synchronised server time with user device.

Life@IITK

This project is aimed at creating a central portal for all activities going in and around the campus. From club events, election results, academic schedule to a personal calendar, everything will be available on a single platform.

  • Rating:
  • Members: 17
  • Website: lifeiitk.tk
  • Completed: Still a running project

With the plethora of activities going on in the campus, going through facebook pages and mails for any information becomes cumbersome. Student participation in various activities has seen a decline because people fail to go through their Facebbok feed or the mails.This application provides all the needs of person ranging from providing various activites, academic schedule, mess menus and much more.

our project objectives was to ensure no more spamming of inbox with irrelevant mails, get event notifications and easily schedule new events, Academic time table and other academic info at hand, Checking mess menu from anywhere anytime,modifying student search to add student POR's and integrating it to our app, Campus map with live event details.
